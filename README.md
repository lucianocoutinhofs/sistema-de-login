# 🔐 Sistema de Login

Sistema de autenticação web desenvolvido em Python com Flask, SQLite e bcrypt.

## 🚀 Tecnologias utilizadas

- **Python** — linguagem principal
- **Flask** — framework web para criar as rotas e o servidor
- **SQLite** — banco de dados para armazenar os usuários
- **bcrypt** — criptografia de senhas
- **HTML + CSS** — interface visual das telas

## ✅ Funcionalidades

- Cadastro de novos usuários
- Login com autenticação segura
- Senhas criptografadas com bcrypt
- Bloqueio após 3
